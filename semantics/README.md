# Shared Maude semantics

Add the shared Maude semantics file here:

```text
semantics.maude
```

The model files load this file through relative paths such as:

```maude
load ../semantics/semantics.maude
```

or

```maude
load ../../semantics/semantics.maude
```

The exact Maude version, commands, and expected output format should be added by Carlos.
